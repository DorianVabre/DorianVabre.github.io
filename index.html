Roborally

This is a Roborally board game I made in a group of 2 during my first year at the UIT of Valence. We used Java and Swing for the graphical interface. The game was fully playable using the console, but the UI to select the cards was not finished by the deadline.

Java, Swing, Board game

OSM117

An OpenStreetMap-based JavaFX application that can be used to find the shortest way between streets of a city. Uses real data from OSM API, uses Speechbrain for voice recognition and authentification.

JavaFX, OpenStreetMap API, Speechbrain API

TinTube

An Android app inspired by Tinder and Youtube. The user can search for a theme, watch videos from Youtube, then swipe if he liked the video or not. A collection of the favorites videos is stored for future uses, where the user can see details about the videos.

Android, API

For want of a sunset when the sky is gone

A 4-days project made by a team of 9 students with Unity (credits in the video). Made with Unity, inspired by "Le petit prince". The controller is a rope that the player has to pull to move deeper in the well.
